# Contributing to Maji Ndogo Water Crisis Analysis

Thank you for your interest in this project! This repository is primarily an educational portfolio project, but feedback and suggestions are always welcome.

## About This Project

This project was created as part of the **ALX Africa Data Analytics curriculum**. The dataset is proprietary to ALX Africa and cannot be shared or redistributed.
